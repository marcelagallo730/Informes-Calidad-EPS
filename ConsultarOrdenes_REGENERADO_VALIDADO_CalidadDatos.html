🚨 Hallazgos Críticos y Problemas Identificados
Validez · Criticidad Media
Dosis = 0 en registros activos
76 registros presentan dosis igual a cero. Una dosis nula en una orden de medicamento activa es clínicamente inválida y representa un error de captura o una omisión en el proceso de prescripción. Estos registros deben ser marcados como erróneos, rechazados en origen o investigados para corrección.
💡 Recomendaciones de Gobernanza de Datos
  • 1
    Implementar regla de validación para dosis: Dosis = 0 debe ser rechazada en el sistema de prescripción. Establecer regla: dosis > 0 AND dosis IS NOT NULL. Para valores superiores a 100, implementar alerta de revisión clínica con validación cruzada contra el campo codigoExterno y el cuadro de medicamentos del PBS.
  • 2
    Estandarizar y tipar el campo telefono: Convertir a tipo string (no numérico) para preservar ceros iniciales. Aplicar validación: exactamente 10 dígitos, prefijo 3xx para celular o 6xx/1xx para fijo. Tratar la columna como NULLABLE en el modelo de datos y eliminar el almacenamiento de strings vacíos como nulos implícitos.
